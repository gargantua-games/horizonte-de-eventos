# premissa
Numa corrida contra o tempo, dois astronautas devem consertar a sua nave antes de chegarem ao ponto sem retorno de um buraco negro. Velocidade, sincronia e comunicação serão indispensáveis enquanto os jogadores enferentam desafios como alienígenas hostis, falta de oxigênio, gravidade zero e asteroides, tudo enquanto coletam recursos para o conserto.
